# Intent Coder

A VS Code extension that converts small, explicit programmer intents into correct, language-specific code at the cursor.

## Features
- **Line Intent → Snippet**: Type a short phrase or task description and press Enter.
- **Selection → Action Menu**: Press `Ctrl+Enter` to optimize, comment, explain, or transform selected code.
- **Template Library**: Insert common competitive programming boilerplate (fast IO, loops, data structures).
- **Language Switching**: Switch languages and re-resolve intents easily.
